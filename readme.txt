== Description ==

Increases number of supported protocols for BuddyPress's users links.
For now additional supported protocols are only apt (ubuntu) and magnet (torrent), but suggestions are welcome :D

== Installation ==

1. This plugin requires BuddyPress (of course).
2. You can download and install "BuddyPress More Protocols" using the built in WordPress plugin installer. If you download "BuddyPress More Protocols" manually, make sure it is uploaded to "/wp-content/plugins/buddypress-more-protocols/".
3. Activate BuddyPress in the "Plugins" admin panel (not network) using the "Activate" link.
4. Enjoy :)
